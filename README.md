# 🎬 CineGuru – Intelligent Movie Information Extractor

CineGuru is an AI-powered Streamlit application that extracts structured movie information from unstructured text using Large Language Models (LLMs). Simply paste a movie-related paragraph, and CineGuru automatically identifies key details such as the movie title, release year, genres, director, cast, rating, and summary.

## ✨ Features

* Extracts movie details from natural language text
* Structured output using Pydantic models
* Powered by Mistral AI and LangChain
* Clean and interactive Streamlit interface
* JSON output validation for transparency
* Fast and accurate information extraction

## 🛠️ Tech Stack

* Streamlit
* LangChain
* Mistral AI
* Pydantic
* Python
* dotenv

## 🚀 How It Works

1. Paste a movie description or paragraph.
2. Click **Extract Movie Insights**.
3. The application processes the text using an LLM.
4. Structured movie information is displayed in an intuitive dashboard.

## Example Input

"3 Idiots (2009) is an Indian Hindi-language coming-of-age comedy-drama film directed by Rajkumar Hirani and starring Aamir Khan, R. Madhavan, and Sharman Joshi."

## Example Output

* Title: 3 Idiots
* Release Year: 2009
* Genre: Comedy, Drama
* Director: Rajkumar Hirani
* Cast: Aamir Khan, R. Madhavan, Sharman Joshi
* Summary: A story about friendship, education, and following one's passion.

Built with ❤️ using LangChain, Mistral AI, and Streamlit.
